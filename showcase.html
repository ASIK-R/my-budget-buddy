Enhanced Bottom Navigation

Square-shaped icons that fill with the primary color on hover, providing a modern and intuitive navigation experience.

Key Features

  • Square-shaped icons with consistent sizing
  • Icons fill with primary color (#076653) on hover
  • Active state shows filled icon with white text
  • Rounded corners for modern appearance
  • Haptic feedback on tap for mobile devices
  • Fully responsive design
  • Smooth transitions for all interactive states

Implementation

The bottom navigation bar uses custom CSS classes defined in index.css for consistent styling:

<button class="bottom-nav-button active">
  <div class="bottom-nav-icon-container">
    <Icon class="bottom-nav-icon active" />
  </div>
  <span class="bottom-nav-label active">Label</span>
</button>

Color Palette

Brand colors used in the navigation:

Primary: #076653
Accent: #E3EF26
